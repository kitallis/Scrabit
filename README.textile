h2. About

p. A Ruby-GNOME applet, that allows you to grab screenshots in less than 3 clicks and/or less than 2 keystrokes onto the scrabit server and copy the urls for sharing directly into your clipboard.

h2. How to use

p. You need to first install :

* ImageMagick
* Curb (gem)
* Ruby-Gnome2

p. Then change into the Source directory and :

* ruby1.8 scrabit.rb
